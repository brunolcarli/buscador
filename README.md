# Buscador de strings em arquivo

Software de busca que retorna string de um arquivo para consultas;

Este programa foi feito para facilitar a busca no Arquivo Morto da Escola, onde é inserido o nome do aluno para que se retorne a linha do arquivo em que se encontra o nome inserido. Nesta linha consta o nome do aluno e o código de localização de sua pasta no Arquivo Físico da Escola. Os computadores da instituição rodam SO Linux possuindo Python 2.5.2 como linguagem nativa, o programa é facilitador ao meio em que se digita o comando 'python busca.py' (sem aspas) no terminal, que irá retornar um pedido de input, que deverá ser o nome do aluno entre 'aspas';

Por eventuais formas de sigilo não será disponibilizado aqui o arquivo contendo os nomes e códigos dos alunos, bem como não será divulgado o nome da instituição, cabendo ao usuário criar ou inserir seu próprio arquivo database para consultas;

Este script é de de livre utilização para qualquer que deseje utilizar, implementar e divulgar no âmbito de promover soluções rápidas e faceis de acesso à dados e informações;

Obs: No caso de utilização do programa, deverá ser criado um arquivo database de extensão .txt ou .csv contendo as linhas a serem buscadas. No caso de utilização por outra instituição de ensino para os mesmos fins, recomenda-se criar uma cópia do arquivo de banco de dados do arquivo morto (neste caso) nos formatos anteriormente citados (.txt/.csv) e coloca-lo junto ao arquivo 'busca.py'.
